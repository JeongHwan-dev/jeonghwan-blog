# Jeonghwan's Blog

웹 개발 지식과 실무 경험을 나누는 테크 블로그입니다. 개발 과정에서 얻은 인사이트와 최신 웹 기술 트렌드 등을 공유합니다.

## 🛠️ 기술 스택

### Frontend

- **코어**: Next.js
- **언어**: TypeScript
- **상태 관리**: TanStack Query
- **스타일링**: Tailwind CSS
- **패키지 매니저**: npm

## 📁 프로젝트 구조

```
src/
├── app/                  # Next.js App Router 페이지
│   ├── _components/      # 페이지별 컴포넌트
│   ├── api/              # API 라우트
│   ├── articles/         # 블로그 글 상세 페이지
│   └── globals.css       # 전역 스타일
├── components/           # 재사용 가능한 컴포넌트
│   ├── ui/               # 기본 UI 컴포넌트
│   ├── features/         # 기능별 컴포넌트
│   ├── layouts/          # 레이아웃 컴포넌트
│   └── themes/           # 테마 관련 컴포넌트
├── lib/                  # 유틸리티 함수
├── services/             # 외부 서비스 연동
├── constants/            # 상수 정의
└── types/                # 타입 정의
```
